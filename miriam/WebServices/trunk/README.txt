MIRIAM Web Services


VERSIONS:
  15 April 2008 (new URI scheme, a lot of method becomed deprecated)
  25 June 2007 (previous version)


INFORMATION:
  Source code of MIRIAM Web Services.
  For more information: http://www.ebi.ac.uk/miriam/


BUILD:
  Two Ant files are provided (build-demo.xml and build-main.xml) to compile the project, according to the version wanted (demo or main).


DEPENDENCIES:
  This software uses Apache Axis and other libraries (cf. the lib folder).

JAVA:
  Needs Java 1.5
