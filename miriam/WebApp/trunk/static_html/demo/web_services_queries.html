Details of the queries

checkRegExp

Checks if the identifier given follows the regular expression of the data type (also provided).

Returns
string - "true" if the identifier follows the regular expression, "false" otherwise
Parameters Type Description
identifier string internal identifier used by the data type
datatype string name, synonym or MIRIAM URI of a data type

getMiriamURI

Transforms a MIRIAM URI into its official equivalent (to transform obsolete URIs into current valid ones). The parameter can be an obsolete URI (URN or URL), but the returned one will always be a URN. This process involve a percent-encoding of all reserved characters (like ':').

Returns
string - the official URI corresponding to the deprecated one, example: "urn:miriam:obo.chebi:CHEBI%3A17891"
Parameter Type Description
uri string deprecated URI (URN or URL), example: "http://www.ebi.ac.uk/chebi/#CHEBI:17891"
