News

2008:
  • April:
    • New MIRIAM URIs released!
    • Web Services updated (don't forget to download the latest Java Library).
    • XML export upgraded: now contains examples of annotation for several formats (such as SBML, BioPAX or CellML).
  • March:
    • New BioModels.net qualifier: "occursIn". For example: "MAP-kinase cascade occursIn human". For more information, cf. qualifiers section.
    • A curator, Nick Juty, has been hired to work on MIRIAM Resources and the Systems Biology Ontology, as part of the European project ELIXIR.
    • Pieces of documentation for a data type can now be stored using a DOI (as well as a PMID or a physical location). Don't forget to reload the cache of your browser (Ctrl+Shift+R under Mozilla Firefox) in order to have access to this new feature.
    • Several updates and modifications are planned over the next months, in order to keep track you can subscribe to our new RSS feed.
  • January:
    • Data type added: SABIO-RK Reaction.
    • The VCell Modeling and Simulation Framework uses MIRIAM Resources.
    • A discussion has started about the URI scheme used in MIRIAM Resources: cf media section.
    • Super-hackathon Standards and ontologies for Systems Biology, 28th January to 2nd February 2008, Okinawa, Japan: cf media section.
2007:
  • December: MIRIAM Resources: tools to generate and resolve robust cross-references in Systems Biology paper published in BMC Systems Biology, cf. media section.
  • November: Post Opening: Senior Database & Ontology Curator - ELIXIR, cf. job Opportunities.
  • October:
    • MIRIAM Resources: robust cross-references in Systems Biology poster presented at the Eighth International Conference on Systems Biology, cf. media section.
    • New annotations: display of the possible elements of a format (currently only information for SBML, CellML and BioPAX is available) which can have a MIRIAM URI associated to a specific data type, for example the possible annotations using Enzyme Nomenclature.
    • Two new BioModels.net qualifiers have been added.
  • July: Semantic annotation of kinetic models in Systems Biology: poster presented at the 15th Annual International Conference on Intelligent Systems for Molecular Biology, cf. media section.
  • April: Dates of creation and last modification added for each data type.
  • March: Several new data types added: WormBase, Pfam, Nucleotide Sequence Database.
  • February: XML Schema available on the export page.
  • January:
    • New version of the Web application (following the new design of the EBI Website).
    • January: The E-MeP project uses MIRIAM Resources.
2006:
  • October: MIRIAM Standard and Resources poster presented at the Seventh International Conference on Systems Biology, cf media section.
  • July: The export from SBML to BioPax (used in BioModels Database) uses the Web Services provided by MIRIAM Resources.
  • June: BioModels Database uses MIRIAM Resources.
