Here is the list of all the services available.

We will be pleased to add any other queries that you would need.

List of available queries

  1. getDataEntries
    Retrieves the physical locationS (URLs) of web pageS about the data-entry.
  2. getDataEntries
    Retrieves the physical locationS (URLs) of web pageS about the data-entry.
  3. getDataEntry
    Retrieves the physical location (URL) of a web page (using a specific resource) about the data-entry.
  4. getDataResources
    Retrieves all the physical locations (URLs) of the services providing the data-type (web page).
  5. getDataTypeAllURIs
    Retrieves all the URIs (URNs and URLs) of the data-type.
  6. getDataTypeDef
    Retrieves the definition of the data-type.
  7. getDataTypePattern
    Retrieves the pattern (regular expression) of the data-type.
  8. getDataTypeSynonyms
    Retrieves all the synonym names of the data-type, with the original name.
  9. getDataTypeURI
    Retrieves the unique (official) URL or URN of the data-type (example: http://www.taxonomy.org/, urn:lsid:uniprot.org).
  10. getDataTypeURIs
    Retrieves all the URLs or URNs of the data-type (examples: http://www.taxonomy.org/, urn:lsid:uniprot.org) including all the deprecated ones.
  11. getDataTypeURL
    Retrieves the unique (official) URL (not a physical URL but a URI) of the data-type (example: http://www.taxonomy.org/).
  12. getDataTypeURLs
    Retrieves the unique (official) URL (not a physical URL but a URI) of the data-type (example: http://www.taxonomy.org/) and all the deprecated ones.
  13. getDataTypeURN
    Retrieves the unique URN of a data-type (example: urn:lsid:uniprot.org).
  14. getDataTypeURNs
    Retrieves the unique (official) URN of the data-type (example: urn:lsid:uniprot.org) and all the deprecated ones.
  15. getDataTypesName
    Retrieves the list of names of all the data types available.
  16. getDataTypesId
    Retrieves the list of identifiers (stable and perenial) of all the data types.
  17. checkRegExp
    Checks if the identifier given follows the regular expression of the data type (also provided).
  18. getName
    Retrieves the common name of a data-type.
  19. getNames
    Retrieves all the names (with synonyms) of a data-type.
  20. getOfficialURI
    Retrieves the offical URI (URN if the user entered a URN, URL if not) of a data-type corresponding to the deprecated one.
  21. getResourceInfo
    Retrieves the general information about a precise resource of a data-type.
  22. getResourceInstitution
    Retrieves the institution which manages a precise resource of a data-type.
  23. getResourceLocation
    Retrieves the location of the servers of a location.
  24. getURI
    Retrieves the unique URI (URL or URN) of the data-entry (example: http://www.geneontology.org/#GO:0045202, urn:lsid:uniprot.org).
  25. getURL
    Retrieves unique URL of the data-entry (example: http://www.geneontology.org/#GO:0045202).
  26. getURN
    Retrieves the unique URN of the data-entry (example: urn:lsid:uniprot.org:P62158).
  27. isDeprecated
    Says if a URI of a data-type is deprecated.

Queries in detail

getDataTypeURN

public java.lang.String getDataTypeURN(java.lang.String name)
Retrieves the unique URN of a data-type (example: urn:lsid:uniprot.org).
Parameters:
name - name primary name (or synonym) of a data-type (examples: ChEBI, UniProt, GO)
Returns:
unique (official) URN of the data-type

getDataTypeURNs

public java.lang.String[] getDataTypeURNs(java.lang.String name)
Retrieves the unique (official) URN of the data-type (example: urn:lsid:uniprot.org) and all the deprecated ones.
Parameters:
name - name or synonym of a data-type (examples: ChEBI, UniProt)
Returns:
unique URN and all the deprecated ones

getDataTypeURL

public java.lang.String getDataTypeURL(java.lang.String name)
Retrieves unique (official) URL (not a physical URL but a URI) of the data-type (example: http://www.taxonomy.org/).
Parameters:
name - name of a data-type (examples: ChEBI, UniProt)
Returns:
unique URL of the data-type

getDataTypeURLs

public java.lang.String[] getDataTypeURLs(java.lang.String name)
Retrieves the unique (official) URL (not a physical URL but a URI) of the data-type (example: http://www.taxonomy.org/) and all the deprecated ones.
Parameters:
name - name of a data-type (examples: ChEBI, UniProt)
Returns:
unique URL of the data-type and all the deprecated ones

getDataTypeURI

public java.lang.String getDataTypeURI(java.lang.String name,
                                       java.lang.String type)
Retrieves the unique (official) URL or URN of the data-type (example: http://www.taxonomy.org/, urn:lsid:uniprot.org).
Parameters:
name - name of the data-type (examples: ChEBI, UniProt)
type - type of the URI the user wants to recover ('URN' or 'URL')
Returns:
unique URL or URN of the data-type

getDataTypeURIs

public java.lang.String[] getDataTypeURIs(java.lang.String name,
                                          java.lang.String type)
Retrieves all the URLs or URNs of the data-type (examples: http://www.taxonomy.org/, urn:lsid:uniprot.org) including all the deprecated ones.
Parameters:
name - name of the data-type (examples: ChEBI, UniProt)
type - type of the URI the user wants to recover ('URN' or 'URL'), can be empty (URL is used in this case)
Returns:
all the URIs (URLs or URNs) of the data-type including all the deprecated ones

Returns:
all the URIs (URLs and URNs) of the data-type including all the deprecated ones
