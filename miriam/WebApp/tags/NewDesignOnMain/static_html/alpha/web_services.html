Miriam Web Services

MIRIAM database provides some web services for programmatic access.

WSDL

You can access to the WSDL file here.

Java Library

In order to communicate with the web services of Miriam in a Java program, you can use the library below (distributed under the terms of the GNU General Public License). Two versions are provided:

  • simple version (you need a couple of external jars to use it)
  • standalone version (all the dependencies are included in the jar)

Here are the dependencies:

  • axis
  • commons-discovery
  • commons-logging
  • jaxrpc
  • log4j
  • saaj
  • wsdl4j
  • activation
  • mail

Note: you can find the latest version of each of these packages on their official web site.

Documentation

The documentation (javadoc) can be found here.

Tutorial

Here is a sample program to show how to use the library:

import uk.ac.ebi.miriam.lib.*;

public class WebServicesTesting
{
  public static void main(String[] args)
  {
   // creation of the link to the web services
   MiriamLink link = new MiriamLink();

   // changes the default address
   ws.setAddress("http://res-022:9951/compneur-srv/miriamws-alpha/MiriamWebServices");

   System.out.println("Test 0:");
   System.out.print("Is 'http://www.ebi.ac.uk/IntEnz/' deprecated? ");
   System.out.println(link.isDeprecated("http://www.ebi.ac.uk/IntEnz/"));
   System.out.println("Official URI: " + link.getOfficialURI("http://www.ebi.ac.uk/IntEnz/"));

   System.out.println("Test 1:");
   System.out.print("Is 'http://www.who.int/classifications/icd/' deprecated? ");
   System.out.println(link.isDeprecated("http://www.who.int/classifications/icd/"));
   System.out.println("Official URI: " + link.getOfficialURI("http://www.who.int/classifications/icd/"));
  }
}
